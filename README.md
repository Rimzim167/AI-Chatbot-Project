# AI Chatbot Project 🤖

This is a simple AI chatbot built using Python and OpenRouter API.

## 🚀 Features
- Chat with AI in terminal
- Uses API for intelligent responses
- Simple and beginner-friendly

## 🛠 Tech Stack
- Python
- Requests Library
- OpenRouter API

## ▶ How to Run

1. Clone repo
2. Create virtual environment
3. Install requirements:

pip install -r requirements.txt

4. Run:

python chatbot.py

## ⚠️ Note
Add your API key in chatbot.py before running.
